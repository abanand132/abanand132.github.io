Password Manager

Password Manager is a application that helps you to manage your passwords efficiently by saving it securely in your device's storage and you can search your password whenever you need them. People rely on third party services to save their password but not anymore.

Features

  • Few personal info required to maintain user's privacy.
  • Advanced encryption techniques is implemented to ensure user's data security.
  • User's data will be stored locally, enchancing user's trust on the software.
  • Password Generator - It comes with built-in password generator which generates strong passwords for the user.
  • Themes : Easy switching between Light/Dark Modes.
  • Easy management of passwords with clean interface.

Platform Supported :

Windows 10 & above
